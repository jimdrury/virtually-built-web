# Virtually Built — Component Studio

Storybook is the workspace for implementing React components from the Pencil design file.

## Workflow

1. **Design in Pencil** — Components live in [`designs.pen`](../../designs.pen). See [components.md](../../docs/components.md) for the full catalog and node links.
2. **Implement in `src/components/`** — Co-locate each component with its `.stories.tsx` file.
3. **Preview in Storybook** — Run `pnpm storybook` and iterate on variants, states, and responsive layouts.
4. **Compose in the app** — Import finished components into Next.js pages under `src/app/`.

Icons use **react-icons** — import from subpaths like `react-icons/lu` at the callsite. See [components.md](../../docs/components.md#icons).

## Directory structure

| Folder | Contents |
| --- | --- |
| `src/components/chrome/` | Site logo, header, footer |
| `src/components/home/` | Hero, episodes section, topics, about, subscribe |
| `src/components/episodes/` | Cards, pagination, theatre stage, transcript |
| `src/components/ui/` | shadcn primitives (buttons, inputs, etc.) |
| `src/components/foundations/` | Tokens, typography, icons, shared utilities |

## Commands

```bash
pnpm storybook          # Dev server at http://localhost:6006
pnpm build-storybook    # Static build to storybook-static/
pnpm test:storybook     # Run story tests via Vitest + Playwright
```
