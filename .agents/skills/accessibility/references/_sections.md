# Sections

Defines example categories, ordering, and filename prefixes.

---

## 1. Visual (visual-)

**WCAG focus:** Perceivable — text sizing, spacing, reflow, contrast  
**Description:** How visual design choices affect users who zoom, change text size, or use high-contrast modes.

## 2. Keyboard (keyboard-)

**WCAG focus:** Operable — focus order, focus appearance, keyboard traps  
**Description:** Keyboard-only and switch-control navigation.

## 3. Semantics (semantics-)

**WCAG focus:** Robust — name, role, value; native HTML over ARIA  
**Description:** Elements, labels, landmarks, and live regions. *(No examples yet.)*

## 4. Motion (motion-)

**WCAG focus:** Operable — reduced motion, vestibular disorders  
**Description:** Animation, autoplay, and `prefers-reduced-motion`. *(No examples yet.)*
